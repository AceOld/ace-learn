总览 · Mission · Lesson 01 / 量化数学体系

收益,是一个随机变量

用概率与统计的语言,重新描述"涨跌" —— 整座量化大厦的第一块砖。

节奏:约 30 分钟 · 适配:理工科本科数学基础(概率统计,可能生疏需激活)

你可能听过无数人说"这只股票会涨"。但量化交易者不说"会涨",他们说:

下一期的收益率 $R$,是一个随机变量,它服从某个我们尚不知道、但可以估计的概率分布。

这一句话,把"猜测"变成了"建模"。整个量化数学体系,本质都是在回答两个问题:

  1. 这个分布长什么样?(均值在哪、尾巴有多厚)
  2. 它和其他资产的分布如何耦合?(相关性、协方差)

本课先把第一块砖铺好:如何定义收益率,以及如何描述它。

本课产出
  1. 理解简单收益 vs 对数收益的本质差异和适用场景
  2. 把"均值/方差"翻译成金融语境的"预期收益/波动率"
  3. 用 Python 跑出"均值难估、波动率易估"的铁证
  4. 记住两个年化换算公式:收益 ×252,波动 ×√252

1. 收益率的两种定义

设资产在时间 $t$ 的价格为 $P_t$。常用两种收益率:

简单收益(simple / arithmetic): $$R_t = \frac{P_t - P_{t-1}}{P_{t-1}} = \frac{P_t}{P_{t-1}} - 1$$

对数收益(log / continuously compounded): $$r_t = \ln\!\left(\frac{P_t}{P_{t-1}}\right) = \ln P_t - \ln P_{t-1}$$

这两者几乎相等,但不等价。它们的差别决定了你什么时候用哪个:

性质简单收益 $R$对数收益 $r$
可加性(跨期)不可加,需连乘可加:$r(0{\to}T) = \sum_t r_t$
可加性(跨资产)组合收益是 $R$ 的加权平均不是简单平均
取值范围$R > -1$(不破产)$(-\infty, +\infty)$,接近正态
建模时组合管理、业绩归因时间序列建模、统计检验
经验法则

跨资产(算组合)用简单收益,跨时间(建模)用对数收益。 这是量化里最常被忽略、却会默默制造 bug 的细节。

2. 把收益当作随机变量来描述

既然 $r$ 是随机变量,我们用统计量刻画它。对于样本 $r_1, \dots, r_n$:

样本均值 $\hat{\mu} = \frac{1}{n}\sum_t r_t$   |   样本方差 $\hat{\sigma}^2 = \frac{1}{n-1}\sum_t (r_t - \hat{\mu})^2$

这两个数字本身平淡无奇,但在金融语境里它们有两个响亮的名字:

统计量金融含义通常量级(A股日线)
均值 $\mu$预期收益(drift / 漂移)极小,接近 0
标准差 $\sigma$波动率(volatility / 风险)~1–2%/日
关键直觉

在日频尺度上,均值(信号)比标准差(噪声)小 1–2 个数量级。 $\mu/\sigma \approx 0.05{-}0.1$。这就是为什么短期预测"涨跌"几乎不可能 —— 噪声比信号大十倍。量化赚钱,靠的是把这个微弱信号在大量独立下注上累积(Law of Large Numbers)。

3. 用 Python 看一眼真实分布

理论说够多了,动手跑。下面用真实风格的数据把上面的概念跑出来(此处用模拟对数收益;后续课程会换成真实行情):

import numpy as np

rng = np.random.default_rng(42)
mu, sigma = 0.0005, 0.015          # 日均 0.05%, 日波动率 1.5%
n = 252 * 5                         # 5 年的交易日
r = rng.normal(mu, sigma, n)        # 对数收益 ~ N(mu, sigma^2)

price = 100 * np.exp(np.cumsum(r))  # 价格 = 100 * exp(累积对数收益)

print("真值年化均值  :", round(mu * 252, 4))                      # 0.126
print("样本年化均值  :", round(r.mean() * 252, 4))                # 0.027
print("样本年化波动率:", round(r.std(ddof=1) * np.sqrt(252), 4))  # 0.235
输出
真值年化均值  : 0.126
样本年化均值  : 0.027
样本年化波动率: 0.235
注意!真值 0.126,样本只估出 0.027

真实漂移率年化 12.6%,但拿 5 年真实数据反推,样本均值只给出 2.7% —— 差了快 5 倍。 波动率却估得很准(真值 $1.5\% \times \sqrt{252} = 23.8\%$,样本 $23.5\%$)。 这不是 bug,正是本课核心结论的铁证:均值(信号)极难估准,标准差(噪声)很容易估准。 这也是为什么"预测涨跌"如此困难,而量化策略更依赖波动率结构而非方向预测。

两个换算公式,记住

年化波动率 = 日波动率 $\times \sqrt{252}$   (平方根法则,来自独立同分布的方差可加性)

年化(对数)收益 = 日均 $\times 252$

4. 检查理解

某策略过去 3 个月日均对数收益 0.02%,日波动率 1.2%。它的年化夏普比率(无风险利率 0)约为?
  • 0.02 / 1.2 ≈ 0.017
  • (0.02% × 252) / (1.2% × 252) ≈ 0.017
  • (0.02% × 252) / (1.2% × √252) ≈ 0.265
  • (0.02% × √252) / (1.2% × √252) ≈ 0.017
答案:C。年化收益 $0.02\% \times 252 = 5.04\%$;年化波动 $1.2\% \times \sqrt{252} \approx 19.05\%$;夏普 $= 5.04/19.05 \approx 0.265$。分子 ×252,分母 ×√252,所以净效果是 ×√252。A/B/D 都错在年化方式上:波动用 √252(因为方差可加),均值用 252(因为期望线性)。
你要计算一个由 60% 股票 + 40% 债券组成的组合的"当日收益",应该用?
  • 对数收益的加权平均
  • 简单收益的加权平均
  • 先取对数再加权,最后 exp 回来
答案:B。跨资产(算组合)应该用简单收益,因为组合收益 $R_p = \sum_i w_i R_i$,这个线性可加性是 Markowitz 组合理论的基石。对数收益不满足这个线性叠加性(对数的和不是和的对数)。A 错;C 失去直观解读且数学上没意义。

本课要点

  • 收益是随机变量。量化建模 = 估计并利用它的分布。
  • 跨资产用简单收益(组合可加权平均),跨时间用对数收益(可累加)。
  • 均值 = 预期收益(信号),标准差 = 波动率(噪声);日频信号比噪声小 1–2 个数量级。
  • 年化:收益 ×252,波动 ×√252。夏普净年化系数是 √252。
