Markowitz 均值-方差与有效前沿

有了协方差矩阵 $\Sigma$,我们终于能回答:"给定这些资产,最优组合权重是多少?"

节奏:约 40 分钟

1952 年 Markowitz 用一个简单的数学公式 $w^\top \Sigma w$(L03 推导过)开启了整个现代投资组合理论(MPT)。这是 1990 年诺贝尔经济学奖的核心成果。本课推导有效前沿,然后用 Python 画出来。

本课产出
  1. 理解 均值-方差框架的目标函数
  2. 推导最小方差组合的闭式解
  3. 画出有效前沿和切点组合
  4. Python 闭式解 + 数值优化两条路都跑通

1. 设定:两个目标函数

给 $n$ 个资产,已知期望收益向量 $\mu$($n$ 维)和协方差矩阵 $\Sigma$($n \times n$)。组合权重 $w$($\sum w_i = 1$)。

  • 组合预期收益:$\mu_p = w^\top \mu$(线性)
  • 组合方差:$\sigma_p^2 = w^\top \Sigma w$(L03,二次型)

Markowitz 的洞察:这两个目标互相冲突 —— 想高收益就得多承担风险。有效前沿就是这两个目标的 Pareto 最优集合。

2. 最小方差组合(闭式解)

最简单的问题:什么权重让风险最小?拉格朗日法:

$$\min_w w^\top \Sigma w \quad \text{s.t.} \quad \mathbf{1}^\top w = 1$$

解出来:

$$w_{\min} = \frac{\Sigma^{-1} \mathbf{1}}{\mathbf{1}^\top \Sigma^{-1} \mathbf{1}}$$

注意这个公式完全不用 $\mu$ —— 最小方差组合只看风险结构,不看预期收益。这是为什么它常被用作"不预测收益时"的稳健组合。

3. 有效前沿:扫描目标收益

更一般的问题:给定目标收益 $\mu_{\text{target}}$,找最小风险权重:

$$\min_w w^\top \Sigma w \quad \text{s.t.} \quad w^\top \mu = \mu_{\text{target}}, \ \mathbf{1}^\top w = 1$$

对每个 $\mu_{\text{target}}$ 求解,得到 $(\sigma_p, \mu_p)$ 平面上的一条双曲线 —— 上半部分就是有效前沿。

"有效"是什么意思

下半部分(同样风险、收益更低)是无效的 —— 没有理性投资者会选。上半部分(给定风险收益最大)才是有效前沿。理性投资者只选有效前沿上的点。

4. 切点组合:引入无风险利率

加入无风险利率 $r_f$,投资者可在无风险资产和 risky 组合间任意分配。从 $(0, r_f)$ 出发画一条与有效前沿相切的射线,切点就是切点组合(tangency portfolio)。它有最高的夏普比率。闭式解:

$$w_{\text{tan}} = \frac{\Sigma^{-1}(\mu - r_f \mathbf{1})}{\mathbf{1}^\top \Sigma^{-1}(\mu - r_f \mathbf{1})}$$

切点组合是 CAPM 的种子:如果所有投资者都用 Markowitz,大家会持有同一个切点组合 —— 这个组合就是"市场组合"。这就是 L05 CAPM 的起点。

5. Python:闭式解画前沿

import numpy as np
import matplotlib.pyplot as plt
import yfinance as yf

tickers = ["SPY", "TLT", "GLD", "EEM", "VNQ"]
px = yf.download(tickers, "2015-01-01", "2024-12-31")["Close"]
rets = np.log(px / px.shift(1)).dropna()

mu = rets.mean().values * 252          # 年化预期收益
Sigma = rets.cov().values * 252        # 年化协方差
n = len(tickers)
ones = np.ones(n)
rf = 0.02                              # 假设无风险利率 2%

# 切点组合(最高夏普)
excess = mu - rf * ones
w_tan = np.linalg.solve(Sigma, excess)
w_tan /= w_tan.sum()

# 最小方差组合
w_min = np.linalg.solve(Sigma, ones)
w_min /= w_min.sum()

# 扫描目标收益画前沿
targets = np.linspace(mu.min(), mu.max(), 50)
frontier_sigma = []
for t in targets:
    # 闭式解(两个等式约束)
    A = np.block([[Sigma, np.c_[mu], np.c_[ones]],
                  [np.c_[mu], [[0, 0]], [[1, 0]]],
                  [np.c_[ones], [[1, 0]], [[0, 0]]]])
    # 简化:用数值优化更清晰,见下文
    frontier_sigma.append(None)  # 占位

# 用 scipy 数值解更简单:
from scipy.optimize import minimize
def port_var(w): return w @ Sigma @ w
constraints = [{"type":"eq","fun":lambda w: w.sum()-1},
               {"type":"eq","fun":lambda w: w@mu - t}]
frontier_sigma = []
for t in targets:
    res = minimize(port_var, np.ones(n)/n, constraints=constraints,
                   bounds=[(0,1)]*n)   # 不允许做空
    if res.success:
        frontier_sigma.append(np.sqrt(res.fun))

# 画图
plt.plot(frontier_sigma, targets, "b-", label="有效前沿")
plt.scatter(np.sqrt(w_min @ Sigma @ w_min), w_min@mu,
            c="green", s=80, label="最小方差组合")
plt.scatter(np.sqrt(w_tan @ Sigma @ w_tan), w_tan@mu,
            c="red", s=80, label="切点组合")
plt.xlabel("年化波动率 σ"); plt.ylabel("年化收益 μ")
plt.legend(); plt.title("Markowitz 有效前沿"); plt.show()
⚠ Markowitz 的实战陷阱

闭式解看起来优雅,但实战中问题极大:
① $\mu$ 极难估准(L01 已说)—— 输入微小变动,输出权重剧烈变化(对参数过度敏感);
② $\Sigma$ 估准但估计误差仍会让前沿"假性精确";
③ 闭式解常给极端负权重(大幅做空),实战不可执行。
正因如此,业界用收缩估计(shrinkage)、风险平价(risk parity)、Black-Litterman 等改进。但 Markowitz 的思想是所有这些改进的源头。

最小方差组合 $w_{\min} = \Sigma^{-1}\mathbf{1} / \mathbf{1}^\top \Sigma^{-1}\mathbf{1}$,关键特征是?
  • 依赖 μ,需先估准预期收益
  • 不依赖 μ,只看协方差结构
  • 等于市场组合
  • 等于等权组合
答案:B。最小方差组合的闭式解完全没有 $\mu$,只依赖 $\Sigma$。这是它的最大优点 —— 回避了 L01 揭示的"均值极难估准"问题。所以实务中,"不预测收益"的稳健策略常用最小方差组合。A 反了;C 是 CAPM 假设下的切点组合;D 等权组合是另一种简化方案。

"Markowitz 闭式解在实战中权重经常剧烈跳动"的主要原因是?
  • 数学推导错误
  • 协方差矩阵不存在逆
  • μ 估不准,微小误差被放大
  • Python 实现问题
答案:C。$w_{\text{tan}} \propto \Sigma^{-1}(\mu - r_f \mathbf{1})$ 中,$\mu$ 的估计误差会被 $\Sigma^{-1}$ 放大。L01 已证明均值极难估准,这正是 Markowitz 实战最大问题。这就是为什么业界发展了 Black-Litterman(把 $\mu$ 用先验+观点混合)、收缩估计(稳定 $\Sigma$)等改进。A/B/D 都不是主因。

本课要点

  • Markowitz 框架:目标 = Pareto 最优(风险 vs 收益),用 $\sigma_p^2 = w^\top \Sigma w$ 度量风险。
  • 最小方差组合:$w_{\min} = \Sigma^{-1}\mathbf{1} / (\mathbf{1}^\top \Sigma^{-1}\mathbf{1})$,不依赖 $\mu$。
  • 切点组合:加入 $r_f$ 后夏普最高的组合,是 CAPM 的种子。
  • 实战陷阱:对 $\mu$ 极敏感,常给极端权重。业界用 Black-Litterman / 收缩 / 风险平价改进。
  • 思想(均值-方差权衡)是所有组合优化的源头。
