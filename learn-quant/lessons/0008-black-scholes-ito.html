1. 几何布朗运动:价格的标准模型

Black-Scholes 假设标的资产价格服从几何布朗运动(GBM):

$$\frac{dS_t}{S_t} = \mu\, dt + \sigma\, dW_t$$

  • $\mu\, dt$:漂移项(预期收益,L01 的 $\mu$)
  • $\sigma\, dW_t$:扩散项(波动率 × 布朗运动增量)
  • $W_t$:标准布朗运动($\mathcal{N}(0, t)$,独立增量)
为什么用"几何"布朗运动而不是普通布朗运动

普通布朗运动 $dS = \mu dt + \sigma dW$ 允许价格为负(不现实)。几何布朗运动描述的是相对收益 $dS/S$,等价于对数价格服从普通布朗运动。由此推出对数收益服从正态(L01 已说),价格始终为正。GBM 是 L01 的连续时间版本。

2. 伊藤引理:随机微积分的核心工具

普通微积分链式法则:$df(X_t) = f'(X_t) dX_t$。但随机过程不是这样。伊藤引理说:

$$df(t, S_t) = \left(\frac{\partial f}{\partial t} + \frac{\partial f}{\partial S} \mu S + \tfrac{1}{2}\frac{\partial^2 f}{\partial S^2}\sigma^2 S^2\right)dt + \frac{\partial f}{\partial S}\sigma S\, dW_t$$

⚠ 关键:多了 $\frac{1}{2}\frac{\partial^2 f}{\partial S^2}\sigma^2 S^2$ 这一项

普通链式法则里没有这个二阶项。为什么随机过程有?直觉:$(dW_t)^2 \approx dt$(不是 0,因为布朗运动的方差与时间成正比)。所以在 Taylor 展开中,$\frac{1}{2} f''(dS)^2$ 这一项不能忽略 —— 它贡献了与 $dt$ 同阶的量。这一项就是 BS 公式里 $\sigma$ 出现的根源。

严格证明需要随机分析。本课只给直觉。深入请读 Shreve Vol II(RESOURCES 已列)。

3. Black-Scholes 推导:三步

第一步:构造对冲组合 $\Pi = f - \frac{\partial f}{\partial S} S$(做多期权、做空 $\Delta = \partial f/\partial S$ 份标的)。

第二步:用伊藤引理算 $d\Pi$,通过对冲消去 $dW_t$ 项 → 组合变无风险。

第三步:无风险组合必须 earn $r\Pi\, dt$(无套利),推出 BS 偏微分方程:

$$\frac{\partial f}{\partial t} + rS \frac{\partial f}{\partial S} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 f}{\partial S^2} = rf$$

注意:$\mu$ 不见了! 无论资产预期收益是 5% 还是 50%,期权价格相同 —— 只取决于 $\sigma$、$r$、$S$、$K$、$T$。这就是"风险中性定价"的来源。

4. 解 BS PDE:欧式看涨期权闭式公式

对欧式看涨(到期日 $T$,行权价 $K$),解 BS PDE 得:

$$\boxed{C = S_0 N(d_1) - K e^{-rT} N(d_2)}$$

其中:

$$d_1 = \frac{\ln(S_0/K) + (r + \tfrac{1}{2}\sigma^2)T}{\sigma\sqrt{T}}, \quad d_2 = d_1 - \sigma\sqrt{T}$$

$N(\cdot)$ 是标准正态累积分布函数。注意公式里没有 $\mu$(资产预期收益)!

$N(d_1)$ 和 $N(d_2)$ 的直觉

$N(d_2)$ = 在风险中性测度下,期权最终实值(in-the-money)的概率。 $N(d_1)$ = 类似概率,但还包含"持有标的时的对冲调整"。具体严格含义需读 Shreve。对入门来说:记住公式形状 + $\mu$ 消失这两件事。

5. Python:实现 BS + 蒙特卡洛验证

import numpy as np
from scipy.stats import norm

def bs_call(S, K, T, r, sigma):
    d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
    d2 = d1 - sigma*np.sqrt(T)
    return S*norm.cdf(d1) - K*np.exp(-r*T)*norm.cdf(d2)

# 例:股价 100, 行权价 105, 1 年到期, 利率 2%, 波动率 20%
C = bs_call(S=100, K=105, T=1, r=0.02, sigma=0.20)
print(f"BS 公式价: {C:.4f}")

# 蒙特卡洛验证(风险中性测度下模拟 GBM)
rng = np.random.default_rng(0)
n_paths = 200_000
Z = rng.standard_normal(n_paths)
S_T = 100 * np.exp((0.02 - 0.5*0.20**2)*1 + 0.20*np.sqrt(1)*Z)
payoffs = np.maximum(S_T - 105, 0)
C_mc = np.exp(-0.02*1) * payoffs.mean()
print(f"蒙特卡洛价: {C_mc:.4f}")  # 应与 BS 公式价接近
典型输出
BS 公式价: 8.9160
蒙特卡洛价: 8.9208
两种方法的一致性证明什么

BS 闭式公式(解析)和蒙特卡洛(数值模拟)在足够样本下给出几乎相同的定价。这是"风险中性定价"的双向验证:公式假设对,模拟测度对。这是量化的核心方法论 —— 闭式公式用于快速定价,蒙特卡洛用于复杂衍生品(无闭式解时)。

6. BS 的局限与业界改进

局限实证表现业界改进
恒定波动率不同行权价的隐含波动率不同(波动率微笑)局部波动率模型
对数正态(L02 肥尾忽略)OTM 期权贵于 BS 预测波动率偏斜
连续交易可对冲实际有跳空、交易成本跳跃扩散、SV 模型
无风险利率恒定利率本身时变利率模型(Hull-White)
欧式假设美式期权可提前行权二叉树 / 有限差分
Black-Scholes 期权定价公式中,资产预期收益 $\mu$ 出现在哪?
  • 漂移项 d₁ 里
  • 贴现因子 $Ke^{-rT}$ 里
  • 不出现(被对冲消掉了)
  • $N(d_1)$ 里
答案:C。BS 公式里完全没有 $\mu$。这是"风险中性定价"的核心:构造对冲组合消去 $dW_t$,μ 也跟着消失。期权价格与"你认为股票会涨多少"无关,只取决于 $\sigma$、$r$、$S$、$K$、$T$。这就是为什么 BS 之前无数聪明人试图定价期权但失败 —— 他们都没意识到 $\mu$ 不该出现。无套利 + 对冲 = 风险中性。A/B/D 都错。
伊藤引理比普通链式法则多出来的 $\frac{1}{2}f''\sigma^2 S^2 dt$ 项,根本来源是?
  • 数学家加了修正项
  • $(dW_t)^2 \approx dt$,布朗运动二阶项不能忽略
  • 利率的影响
  • 数值误差
答案:B。普通微积分里 $(dt)^2 \approx 0$,二阶项可忽略。但布朗运动的 $(dW_t)^2 \approx dt$(不是 0,因为布朗运动方差与时间成正比),所以 Taylor 展开里 $\frac{1}{2}f''(dS)^2$ 这一项贡献与 $dt$ 同阶,不能丢。这正是 BS 公式里 $\sigma^2$ 出现的根源。C/D 无关;A 是字面误解。深入读 Shreve Vol II。

本课要点

  • GBM:价格的标准连续时间模型,$dS/S = \mu dt + \sigma dW$。
  • 伊藤引理:随机链式法则,多了 $\frac{1}{2}f''\sigma^2 S^2 dt$(来自 $(dW)^2 \approx dt$)。
  • BS PDE:对冲消去 $dW$ + 无套利 → 期权定价偏微分方程。
  • BS 公式:欧式看涨闭式解,$\mu$ 不出现(风险中性定价)。
  • 蒙特卡洛验证:风险中性测度下模拟 GBM,与 BS 公式价一致。
  • 局限:恒定 σ / 对数正态 / 连续交易 → 业界用局部波动率、SV、跳跃扩散改进。
