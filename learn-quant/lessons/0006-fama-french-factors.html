Fama-French 三因子回归

CAPM 的 $\alpha$ 显著,是真 alpha 还是漏了因子?Fama-French 用规模和价值两个因子给出答案。

节奏:约 40 分钟

L05 结尾我们说 CAPM 实战 $R^2$ 常低于 0.5,大段收益变化无法被市场因子解释。Fama 和 French 1992 年发现,加上规模因子 SMB 和价值因子 HML 后,大部分 CAPM 的 "$\alpha$" 消失了。这就是著名的三因子模型 —— 因子投资的起点。

本课产出
  1. 理解 SMB / HML 因子的构造逻辑
  2. 会跑三因子回归:$R_i - r_f = \alpha + \beta_m \text{MKT} + \beta_s \text{SMB} + \beta_v \text{HML} + \varepsilon$
  3. 对比 CAPM 单因子和 FF 三因子的 $\alpha$ 变化
  4. 从 Kenneth French 数据库直接拿因子数据复现

1. CAPM 解释不了的两个异象

1980s-90s 实证研究反复发现两个 CAPM 解释不了的规律:

异象现象解释力来源
规模效应小市值股票长期超额收益高于 CAPM 预测SMB(Small Minus Big)
价值效应低估值(高 B/M)股票长期超额收益高于 CAPM 预测HML(High Minus Low)

Fama-French 把这两个异象显式建模为因子,由此衍生出三因子模型。

2. 三因子模型

$$R_{i,t} - r_f = \alpha_i + \beta_{i,m}\text{MKT}_t + \beta_{i,s}\text{SMB}_t + \beta_{i,v}\text{HML}_t + \varepsilon_{i,t}$$

因子构造经济学含义
MKT$R_m - r_f$,市场超额系统风险(L05 CAPM)
SMB小盘股收益 − 大盘股收益规模风险溢价
HML高 B/M − 低 B/M价值风险溢价
为什么是"因子"不是"特征"

因子是对所有资产同时起作用的公共冲击。SMB 不是"小盘股自身的事",是"规模这个风险维度上,所有暴露于此的资产同时受影响"。因子模型的本质是把资产收益投影到少数几个公共风险源上。这就是为什么它能解释 CAPM 解释不了的 $\alpha$。

3. 从 Kenneth French 数据库拿因子数据

Kenneth French 公开维护着免费的标准因子数据(学术界和业界通用)。官方数据库。在 Python 中通过 pandas-datareader 或 ffn 直接拉取。

# 方式 1: pandas-datareader
# pip install pandas-datareader
from pandas_datareader.famafrench import FamaFrenchReader
reader = FamaFrenchReader("F-F_Research_Data_Factors_daily", start="2019-01-01")
factors = reader.read()[0] / 100  # 原始是百分比,转小数
print(factors.head())
# 列:MKT-RF, SMB, HML, RF

# 方式 2: yfinance 拉个股,与因子对齐
import yfinance as yf
import numpy as np
aapl = yf.download("AAPL", "2019-01-01", "2024-12-31")["Close"]
aapl_rets = np.log(aapl / aapl.shift(1)).dropna()
aapl_excess = aapl_rets - factors["RF"]

# 对齐日期
df = factors.join(aapl_excess.rename("AAPL_RF"), how="inner").dropna()

# 三因子回归
import statsmodels.api as sm
X = df[["Mkt-RF", "SMB", "HML"]]
X = sm.add_constant(X)
y = df["AAPL_RF"]
model = sm.OLS(y, X).fit()
print(model.summary())
典型结果(AAPL, FF 三因子)
const (α)   : 0.038 (p=0.41,不显著)
Mkt-RF (βm) : 1.15 (p<0.001)
SMB    (βs) : -0.22 (p=0.04,显著负 → AAPL 偏大盘)
HML    (βv) : -0.45 (p<0.01,显著负 → AAPL 偏成长)

4. 关键观察:CAPM 的 $\alpha$ 被吃掉了

模型$\alpha$(日)$R^2$
CAPM 单因子(L05)0.045(不显著)~0.45
FF 三因子(本课)0.038(更不显著)~0.55
⚠ 因子投资的核心教训

AAPL 的 $\alpha$ 在三因子下进一步缩小且更不显著。这说明 CAPM 框架下"看似有 alpha"的资产,加入 SMB/HML 后被识别为"对规模/价值因子的暴露"而非真 alpha。这是因子投资的核心命题:大部分声称的 alpha,只是尚未识别的 beta。后续还有五因子(加 RMW 盈利、CMA 投资)、动量因子(MOM)、质量因子等 —— 都是同一个逻辑的延伸。

某基金在 CAPM 下 $\alpha = 3\%$(显著),加入 Fama-French 后 $\alpha = 0.2\%$(不显著)。这说明?
  • 基金仍有真 alpha,FF 模型错了
  • CAPM 的 alpha 来自对规模/价值因子的暴露
  • 基金在亏钱
  • 市场无效
答案:B。CAPM 单因子下显著的正 $\alpha$,在加入 SMB/HML 后消失 → 说明该基金的"alpha" 实际上是对规模/价值因子的暴露,不是真正的选股能力。这是因子投资最核心的实证教训。大部分"alpha"只是未识别的"beta"。A 是基金销售话术;C/D 无关。这也是为什么业界做业绩归因时,三因子 / 五因子是标配。
为什么 SMB 和 HML 被称为"因子"而不是股票特征?
  • 因为它们数学公式复杂
  • 因为只在某些行业有效
  • 因为它们是对所有资产同时起作用的公共风险源
  • 因为 Fama-French 是诺贝尔奖得主
答案:C。因子的本质是公共风险源 —— 对所有资产同时起作用的系统性冲击。SMB 不是"小盘股的事",是"规模这个风险维度上,所有暴露于此的资产共同承受的冲击"。这正是因子模型与简单"加特征变量回归"的根本区别。A/B/D 都不触及定义。

本课要点

  • 三因子模型:MKT + SMB(规模)+ HML(价值),解释 CAPM 残差。
  • 因子的本质:公共风险源,对全资产同时作用。
  • 核心教训:大部分"alpha"只是未识别的"beta"。
  • 因子扩展:五因子(+RMW+CMA)、动量(MOM)、质量等。
  • Kenneth French 数据库是免费标准数据源。

推荐主源

本课主源:Fama & French 1993 原始论文 "Common Risk Factors..."。因子投资的开山之作,读完感受当年发现"CAPM 漏因子"的实证冲击。
