波动率与正态性幻觉

上一课说"对数收益接近正态"。本课拆穿这个幻觉:真实收益分布的尾巴,远比你想象的肥。

节奏:约 35 分钟

L01 结尾我们用 rng.normal() 生成模拟收益,假设它服从正态分布。但真实金融数据服从正态吗? 这一课用三个统计量(偏度、峰度、QQ 图)给出铁证:不是,而且差得离谱。这个结论会贯穿后续所有课程,直接影响 VaR、期权定价、组合风险计算。

本课产出
  1. 会算并解读 偏度(Skewness)和峰度(Kurtosis)
  2. 能从 QQ 图一眼看出尾部肥瘦
  3. 理解波动率聚类现象(为 L07 GARCH 铺路)
  4. 知道为什么 1987 黑色星期一在正态模型下"几十亿年一次"

1. 正态分布的承诺

正态分布 $\mathcal{N}(\mu, \sigma^2)$ 有两个参数:

  • $\mu$:均值(L01 已说,对应"预期收益")
  • $\sigma^2$:方差(对应"波动率平方")

它承诺了一件量化交易者梦寐以求的事:只用两个数字就能完整描述收益分布。如果这是真的,风险计算、组合优化、期权定价全部可以用闭式公式精确算出。问题是 —— 它不是真的。

2. 偏度与峰度:戳破幻觉的两把刀

正态分布的两个隐含性质:

  • 对称(左尾右尾镜像)→ 偏度 = 0
  • 既不瘦不胖的尾巴 → 超额峰度 = 0(原始峰度 = 3)

$$\text{偏度} = \frac{\mathbb{E}[(r-\mu)^3]}{\sigma^3}, \quad \text{峰度} = \frac{\mathbb{E}[(r-\mu)^4]}{\sigma^4}, \quad \text{超额峰度} = \text{峰度} - 3$$

统计量正态预期真实股票收益含义
偏度0< 0(负偏)左尾更长 —— 大跌比大涨更极端
超额峰度0>> 0(常 3–20)肥尾 —— 极端事件频率远高于正态预测
肥尾的实战含义

正态模型会说"日跌幅超过 4σ 的事件,每 15784 年发生一次"。 但 A 股、美股在 2008、2015、2020 反复出现 4σ 以上的单日波动。 正态分布把"几十年一遇"算成"几万年一遇" —— 这就是为什么基于正态的 VaR 在危机时系统性低估风险。

3. Python 实证:看一眼真实收益的高阶矩

import numpy as np
import yfinance as yf           # pip install yfinance
from scipy import stats

# 拉标普 500 近 20 年
sp = yf.download("^GSPC", "2004-01-01", "2024-12-31")["Close"]
r = np.log(sp / sp.shift(1)).dropna()

print(f"样本均值   : {r.mean()*252:.4f}  (年化)")
print(f"波动率     : {r.std(ddof=1)*np.sqrt(252):.4f}")
print(f"偏度       : {stats.skew(r):.4f}")     # 预期 < 0
print(f"超额峰度   : {stats.kurtosis(r):.4f}")  # 预期 >> 0(正态为 0)
典型输出(标普 500 日对数收益,2004–2024)
样本均值   : 0.0728  (年化)
波动率     : 0.1985
偏度       : -0.85
超额峰度   : 13.27
解读

偏度 $-0.85$(显著负偏,大跌更狠);超额峰度 $13.27$(正态应该是 0,实际远超)。这意味着标普的日收益分布尾巴比正态肥得多。如果你用正态模型做风险预算,会系统性低估尾部风险。

4. QQ 图:肉眼判肥尾

QQ 图(Quantile-Quantile)是把样本分位数对着正态分位数画散点。如果数据真服从正态,点会落在一条直线上。

import matplotlib.pyplot as plt
stats.probplot(r, dist="norm", plot=plt)
plt.title("S&P 500 日对数收益 vs 正态 QQ 图")
plt.show()

你会看到:中间段大致直线,但两端(尾部)明显偏离 —— 左下角比直线更深(更极端的负收益),右上角也偏离。这就是肥尾的视觉证据。

某股票收益样本超额峰度 = 8.5,说明?
  • 完全服从正态分布
  • 有显著肥尾,正态模型会低估尾部风险
  • 收益对称,无偏
  • 波动率为零
答案:B。超额峰度 = 0 才是正态;> 0 = 比正态更尖更肥尾(极端事件频率更高)。$8.5$ 是显著肥尾,意味着用正态算 VaR 会系统性低估崩盘概率。A 错(8.5 ≠ 0);C 错(峰度不衡量偏度,偏度另算);D 完全无关。

5. 波动率聚类:第二个幻觉

正态分布还有一个隐含假设:方差恒定(同方差)。但真实金融数据中,波动率明显时高时低 —— 高波动期(如 2008 危机、2020 疫情)会聚集出现,低波动期也成段。这叫波动率聚类(volatility clustering)。

假设正态模型真实市场
方差恒定时变(聚类)
自相关收益本身无自相关收益无,但收益平方有强自相关
建模够用需 GARCH(L07 课)
检验波动率聚类的最简方法

对收益 $r_t$ 算自相关(通常接近 0,支持"价格不可预测")。 但对收益平方 $r_t^2$ 算自相关 —— 如果显著不为 0,说明"大波动跟着大波动",即聚类。这是 L07 GARCH 模型的统计学依据。 

# 检验波动率聚类
import statsmodels.api as sm
print("r  自相关:", sm.tsa.acf(r, nlags=10)[1:].round(3))    # 通常接近 0
print("r² 自相关:", sm.tsa.acf(r**2, nlags=10)[1:].round(3)) # 显著正 → 聚类
某资产收益本身的自相关接近 0,但收益平方的自相关显著为正。这说明?
  • 收益可预测,可以套利
  • 收益完全服从正态
  • 波动率聚类,需 GARCH 建模
  • 数据有问题,应丢弃
答案:C。收益本身无自相关(弱有效市场),但收益平方有强自相关 = 波动率聚类(大波动跟着大波动)。这是 GARCH 模型的核心动机(L07 课详讲)。A 错(收益无自相关 = 不可预测);B 错(正态意味着 $r^2$ 也无自相关);D 错(这是金融数据的普遍特征,不是 bug)。

本课要点

  • 真实收益不是正态:显著负偏 + 肥尾(超额峰度常 3–20)。
  • 正态模型系统性低估尾部风险:4σ 事件本应万年一次,实际几十年一次。
  • QQ 图是肉眼判肥尾的工具:两端偏离直线 = 肥尾。
  • 波动率聚类:收益无自相关,但 $r^2$ 有强自相关 → 需 GARCH(L07)。
  • 所有这些幻觉戳破后,后续课程的 VaR / 期权定价 / 组合风险都会用更现实的分布(t 分布、GARCH、EVT 极值理论)。
