CAPM 与 $\beta$

如果所有投资者都用 Markowitz,市场均衡时单资产预期收益只取决于它对市场组合的贡献 —— 这就是 $\beta$。

节奏:约 35 分钟

L04 结尾说切点组合是 CAPM 的种子。本课把种子发芽:假定所有投资者都用 Markowitz 且持有同一个切点组合,市场出清时这个组合就是"市场组合",单资产的预期收益由此被决定。这就是 CAPM(Capital Asset Pricing Model),Sharpe 1964 年提出,现代金融的奠基模型。

本课产出
  1. 推导 CAPM 主方程:$\mathbb{E}[R_i] = r_f + \beta_i(\mathbb{E}[R_m] - r_f)$
  2. 理解 $\beta_i = \text{Cov}(R_i, R_m)/\sigma_m^2$ 的含义
  3. 会用 OLS 回归估 $\beta$,解读截距 $\alpha$
  4. 知道 CAPM 的三大局限(为 L06 Fama-French 铺路)

1. CAPM 推导:三步

第一步:Markowitz 均衡 → 所有投资者持有切点组合 $M$。

第二步:市场出清 → $M$ 必须等于所有风险资产的市值加权组合。这就是"市场组合"。

第三步:在均衡下,资产 $i$ 的边际风险贡献必须等于它的边际收益补偿。单资产对组合的风险贡献不是 $\sigma_i$,而是它与市场的协方差。由此推出 CAPM 主方程:

$$\boxed{\mathbb{E}[R_i] = r_f + \beta_i\bigl(\mathbb{E}[R_m] - r_f\bigr)}$$

其中:

$$\beta_i = \frac{\text{Cov}(R_i, R_m)}{\sigma_m^2} = \frac{\sigma_{im}}{\sigma_m^2} = \rho_{im}\frac{\sigma_i}{\sigma_m}$$

2. $\beta$ 的三种含义

视角$\beta$ 的解读
数学$R_i$ 对 $R_m$ 的回归斜率(下节 OLS)
风险资产 $i$ 的系统风险(不可分散)贡献
组合加入少量 $i$ 到市场组合时,组合风险的相对增量
为什么是 $\beta$ 而不是 $\sigma$

L03 证明:组合方差随 $n$ 增大趋近于系统风险 $\sigma^2 \bar{\rho}$,非系统风险可被分散掉。 所以市场只对不可分散的部分补偿,即 $\beta$。高 $\sigma$ 但低 $\beta$ 的资产不应获得高收益(因为风险可被分散)。这是 CAPM 的革命性洞察。

3. 估 $\beta$:OLS 回归

从 CAPM 主方程,实际收益可写为:

$$R_{i,t} - r_f = \alpha_i + \beta_i(R_{m,t} - r_f) + \varepsilon_{i,t}$$

用 OLS 回归估 $(\alpha_i, \beta_i)$:$\beta$ 是斜率,$\alpha$ 是截距。如果 CAPM 严格成立,$\alpha$ 应该是 0(没有超额收益)。$\alpha \neq 0$ 就是"超越市场"的证据 —— 这是量化策略的核心目标。

import yfinance as yf
import statsmodels.api as sm

# 拉苹果和市场
data = yf.download(["AAPL", "^GSPC"], "2019-01-01", "2024-12-31")["Close"]
data.columns = ["AAPL", "MKT"]
rets = np.log(data / data.shift(1)).dropna() * 100   # 单位 %

rf_daily = 0.02/252 * 100  # 年 2% 无风险利率折日
X = (rets["MKT"] - rf_daily)        # 市场超额收益
y = (rets["AAPL"] - rf_daily)       # AAPL 超额收益
X = sm.add_constant(X)

model = sm.OLS(y, X).fit()
print(model.summary())
# 看 const(α)和 MKT(β)
典型结果(AAPL 2019-2024)
const (α)  : 0.045 (p=0.32,不显著)
MKT   (β)  : 1.18  (p<0.001,显著)

解读:AAPL 的 $\beta = 1.18$,意味着市场涨 1% 时 AAPL 平均涨 1.18% —— 比市场更具进攻性。$\alpha$ 不显著,意味着这一段时间没有显著超越 CAPM 预测的超额收益。

4. CAPM 的三大局限

局限问题改进方向
单一因子只用市场因子解释收益,实证拟合差Fama-French 三/五因子(L06)
假设过强所有投资者同预期、同持有期、无税套利定价理论 APT
$\beta$ 估计不稳不同窗口估出的 $\beta$ 差异大Bayesian 收缩、行业 $\beta$ 平均
⚠ CAPM 实战怎么用

不要用 CAPM 做精确预测(实证早证明单因子解释力 $R^2$ 常低于 0.5)。用它做粗略资本预算、估值贴现率、相对风险排序。要解释收益,直接用 Fama-French(L06)。但 CAPM 的思想(只补偿系统风险)是所有后续因子模型的根。

某资产 $\beta = 1.5$,市场预期超额收益 6%,无风险 2%。CAPM 预测该资产预期收益?
  • 6% × 1.5 = 9%
  • 2% + 6% = 8%
  • 2% + 1.5 × 6% = 11%
  • 2% × 1.5 = 3%
答案:C。CAPM:$\mathbb{E}[R_i] = r_f + \beta(\mathbb{E}[R_m] - r_f) = 2\% + 1.5 \times 6\% = 11\%$。注意公式用的是市场的超额收益($\mathbb{E}[R_m] - r_f = 6\%$),不是市场收益本身。A 漏了 $r_f$;B 漏了 $\beta$;D 用错了基准。CAPM 的几何直觉:β 是"市场超额收益"的放大系数,起点是 $r_f$。
CAPM 预测 $\alpha = 0$,如果实证回归发现某基金 $\alpha = 2\%$ 且显著,这意味着?
  • CAPM 错了,应丢弃
  • 基金在风险调整后跑赢市场
  • 市场无风险
  • β 算错了
答案:B。$\alpha > 0$ 且显著 = 在控制了市场风险($\beta$)之后,还有不可解释的正超额收益。这是"基金经理真有 alpha"的证据(在 CAPM 框架下)。但 A 也是部分对(CAPM 可能漏了因子),Fama-French(L06)会告诉你 $\alpha$ 是不是被其他因子吃掉了。C/D 无关。

本课要点

  • CAPM:$\mathbb{E}[R_i] = r_f + \beta_i(\mathbb{E}[R_m] - r_f)$,从 Markowitz 均衡推导。
  • $\beta$:系统风险贡献;数学上是 $R_i$ 对 $R_m$ 的回归斜率。
  • $\alpha$:CAPM 框架下不能解释的超额收益,量化策略追求的目标。
  • 三大局限:单因子 / 假设强 / $\beta$ 不稳。
  • 不要用 CAPM 做精确预测,用作粗略风险排序和资本预算。

推荐主源

本课主源:Sharpe 1964 原始论文 "Capital Asset Prices"。读完感受 CAPM 推导的纯粹美感。
